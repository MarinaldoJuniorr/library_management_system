# Library Management System

A local library management system built with Python, Streamlit and Excel.

## Features

### Books
- Create books
- Search books
- Update books
- Delete books
- Automatic label generation
- Availability control

### Users
- Create users
- Search users
- Update users
- Delete users

### Loans
- Register loans
- Availability tracking

### Returns
- Register returns

## Technologies

- Python
- Streamlit
- Pandas
- OpenPyXL
